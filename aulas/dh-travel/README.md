## Objetivo da aula
O objetivo desta aula é entender e praticar sobre:

- Modelos de caixa
  - Width
  - Height
  - Padding
  - Border
  - Margin
  - Box sizing
  
- Elementos de linha e bloco
- Medidas relativas
  - Pixel
  - %
  - Em
  - Rem
  - Viewport

- *Media queries*
